funki-hubot
==============

## What's this ?

「Twitter上のだれかさん」など、「あまり似ていないが、その人の雰囲気に近づけてしゃべる」ことを目的とした雰囲気botです。

具体的には、Redisに蓄えられた「名詞」「形容詞」「定型文」をランダムに組み合わせ、一定時間を置いて話します。
